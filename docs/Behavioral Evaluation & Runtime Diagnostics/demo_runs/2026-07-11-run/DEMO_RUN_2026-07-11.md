## What broke (fix before Run 7)

1. **The value-pump loop** (root): single-URL re-read → timestamp-footer defeats
   content-hash dedup → 387 credited rewrites → commitment value inflates → monopoly.
   The URL-dedup fix is built but uncommitted and untested in a life; it treats the
   re-read, **not** the credit inflation. Normalize volatile footers out of the ledger
   hash and decay repeat-credit per artifact path.
2. **Credit keys off the committed goal, not the content** (R-D live): a *world
   knowledge* blog memo paid `self_understanding` 403 times while
   `world_knowledge` earned 0. Convergence (Fix 4) converged on mis-keyed credit.
3. **`write_exemplar` is structurally dead**: 12× EACCES on the quality-golden
   exemplars dir from minute 13; path is writable post-mortem, cause unresolved —
   needs errno-level logging + a boot writability probe. Until then the quality
   standard can never grow from demonstrated-good work.
4. **Avoidance release needs a re-commit cooldown**: releasing the incumbent means
   nothing if it instantly wins the next commit sort.
5. Reuse/synthesis starved again by the back-half treadmill (reuse 2, syntheses 0,
   funnel still candidate-only).
