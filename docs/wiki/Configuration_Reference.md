# Configuration Reference

This page lists important ORRIN_* configuration variables. Use `pyproject.toml` and environment files for defaults.

Examples
- ORRIN_CYCLE_SECONDS: default cycle duration
- ORRIN_REWARD_SETPOINT: baseline reward setpoint
- ORRIN_BANDIT_EPSILON: exploration parameter
