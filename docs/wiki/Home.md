# Home

Welcome to the Orrin project wiki. This site documents the design, operation, and internals of Orrin — an experimental symbolic-first cognitive agent with affect, goals, memory, and metacognition, plus a live Face & Brain UI.

- Provide an authoritative source of truth for developers, researchers, and operators.
- Explain core concepts, architecture, and how to run, extend, and debug Orrin.
