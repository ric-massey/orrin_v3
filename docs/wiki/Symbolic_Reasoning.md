# Symbolic Reasoning

Overview
- Symbolic reasoning uses explicit representations (graphs, predicates, rules) to perform inference and causal reasoning without LLMs.

World model
- Graph of entities, predicates, and temporal relations. Supports queries and updates.

Causal reasoning
- Causal graphs enable counterfactual reasoning and intervention experiments.

Concept formation & analogies
- Clustering-based abstraction with naming heuristics; analogies via structural mapping algorithms.
