# What is Orrin?

Orrin is an experimental cognitive architecture that places symbolic systems, control signals, and persistent memory at the center of autonomy, using language models as optional tools. Key design goals:
- Symbolic-first cognition: explicit world models, deterministic reasoning, and inspectable control paths.
- Continuous autonomy: a recurring cognitive loop (cycles) coordinates perception, memory retrieval, deliberation, execution, and consolidation.
- Persistent memory and consolidation: short-term working memory and a long-term embedding-backed store maintained by background daemons.
- Observable runtime: the Face & Brain UI provides live visibility into internal state, control signals, and goals.

Who should use Orrin?
- Researchers exploring hybrid symbolic/neural architectures.
- Developers building long-running agents with durable state.
- Operators who need transparent, inspectable autonomous processes.

Limitations
- Experimental codebase; not production hardened.
- Learning and adaptation mechanisms are early-stage; expect tuning.
